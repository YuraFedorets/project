# УКД Talent — Recruitment Platform

Платформа працевлаштування для студентів Університету Короля Данила.

## Структура проекту

```
my-flask-app/
├── static/
│   ├── css/
│   │   └── style.css        # Всі стилі (фон, картки, навігація, форми)
│   └── js/
│       └── script.js        # Вся клієнтська логіка (чати, модалки, пошук)
├── templates/
│   ├── base.html            # Базовий шаблон (navbar, модальні вікна, підключення CSS/JS)
│   └── index.html           # Головний контент (лендінг, рейтинг, запрошення, профіль тощо)
├── app.py                   # Точка запуску Flask-додатку
├── config.py                # Конфігурація (SECRET_KEY, DATABASE, TG_TOKEN тощо)
├── database.py              # Підключення до SQLite, ініціалізація таблиць
├── routes.py                # Всі Flask маршрути (замінює logic.py)
├── bot.py                   # Telegram-бот підтримки
└── requirements.txt         # Залежності
```

## Запуск

```bash
pip install -r requirements.txt
python app.py
```

Сервер запуститься на http://localhost:5000

## Доступ за замовчуванням

- **Логін адміна:** `admin`
- **Пароль:** `123`

## Ролі користувачів

| Роль | Опис |
|---|---|
| `ADMIN` | Повний доступ, управління студентами, компаніями, підтримкою |
| `COMPANY_ADMIN` | Управління своєю компанією, додавання працівників |
| `EMPLOYEE` | Перегляд і видалення запрошень своєї компанії |
| `STUDENT` | Власний профіль, перегляд і відповідь на запрошення |
